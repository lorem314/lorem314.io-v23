## `codehike` 中的 annotation 测试

这篇博客用于查看和测试 `codehike` 中的各种 annotation 的效果。

### callout

```text
// !callout[/amet/] This is a callout
```

使用注释添加这一行，来添加 callout。

```js
const lorem = ipsum(dolor, sit)
// !callout[/amet/] This is a callout This is a callout This is a callout This is a callout This is a callout
const [amet, consectetur] = [0, 0]
```

```js showLineNumbers
const lorem = ipsum(dolor, sit)
// !mark
// !callout[/amet/] This is a callout This is a callout This is a callout This is a callout This is a callout
const [amet, consectetur] = [0, 0]
lorem.adipiscing((sed, elit) => {
  if (sed) {
    amet += elit
  }
})
```

### mark

```text
// !mark(1:2) gold
// !mark[/sit/] pink
```

```js
function lorem(ipsum, dolor = 1) {
  // !mark
  return dolor
}

function ipsum(lorem, dolor = 1) {
  // !mark(1:2) gold
  const sit = lorem == null ? 0 : lorem.sit
  dolor = sit - amet(dolor)
  // !mark[/sit/] pink
  return sit ? consectetur(lorem) : []
}
```

### diff

```text
// !diff -
// !diff +
```

function lorem(ipsum, dolor = 1) {
  const sit = ipsum == null ? 0 : ipsum.sit
  // !diff -
  dolor = ipsum - sit
  // !diff +
